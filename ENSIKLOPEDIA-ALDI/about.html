Pembuat Website


Aldi Pangestu Kurniawan

Hallo semuanya!!! Saya Aldi Pangestu Kurniawan, kelas 10-1, Sekolah Menengah Atas (SMA) Talenta. Website ini saya buat untuk memenuhi nilai sejarah, informatika, dan bahasa Indonesia. Selain itu, pembuatan website ini juga untuk memperbanyak pengalaman saya dalam membuat sebuah website yang nantinya akan berguna di masa yang akan datang. Di website ini saya menjelaskan mengenai Kerajaan Aceh, di dalamnya terdapat mengenai kehidupan politik Kerajaan Aceh (sultan-sultan yang memimpin Kerajaan Aceh), kehidupan sosial (Cara hidup Kerajaan Aceh), peninggalan budaya Kerajaan Aceh, kehidupan ekonomi (aktivitas ekonomi yang dilakukan Kerajaan Aceh), dan kemunduran Kerajaan Aceh (penyebab goyahnya pemerintahan Kerajaan Aceh). Pada pembuatan website ini, hal yang tersulit adalah ketika memasukan kode HTML. Kurangnya pengalaman membuat saya harus mencari informasi tentang bagaimana cara untuk memasukkan kode HTML, CSS, dan framework Bootstrap yang sesuai. Dalam pencarian informasi, saya mencari dari YouTube dan w3school. Walau begitu, saya tetap kesulitan dalam pembuatan web. Penggunaan elemen div yang harus teliti menjadi kesulitan utama saya. Saya juga kesulitan dalam membuat elemen-elemen memasukan gambar yang berkaitan dengan elemen href. Selain itu, saya juga agak kesulitan dalam persyaratan harusnya 300 kata dalam 1 slide yang menambah kesulitan dalam penyusunan atau pembentukan informasi yang dimasukkan. Melalui website ini, ada banyak kemampuan yang saya kembangkan. Seperti, kemampuan coding saya dan juga kemampuan merangkai kata-kata. Saya sadar betul. Melalui tugas ini, kemampuan coding saya meningkat karena dipaksa untuk mengulik atau mencari tahu sendiri cara membuat website yang sesuai dengan kemauan kita. Kemampuan merangkai kata-kata saya juga meningkat karena, dalam pembuatan wesite ini saya dipaksa untuk membuat tiga ratus kata dalam satu halaman. Melalui penugasan ini, saya jadi sadar pentingnya memanfaatkan waktu semaksimal mungkin. Karena dalam pembuatan website saya agak bersantai-santai di awal penugasan menjadi kesulitan di akhir penugasan yang mengharuskan saya untuk mengejar deadline yang waktunya tidak banyak lagi. Jika saya bisa memperbaiki penugasan ini, saya akan menggunakan waktu yang diberikan semaksimal mungkin agar hasil tugas bisa lebih maksimal atau lebih bagus.
